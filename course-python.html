Chapter 1: Introduction to Python

Python is a high-level, interpreted language known for its simplicity and readability.

Example:

print("Hello, Python!")

Exercise:

Print your name using Python.

Chapter 2: Variables and Data Types

Variables store values. Python supports types like int, float, str, and bool.

Example:

name = "Fayda"
age = 18
print(name, age)

Exercise:

Declare two variables: your age and your city, then print them.

Chapter 3: Conditional Statements

Use if, elif, and else to control the flow of your program.

Example:

score = 70
if score >= 60:
    print("Pass")
else:
    print("Fail")

Exercise:

Write a program to check if a number is even or odd.

Chapter 4: Loops

Loops repeat actions. Python supports for and while loops.

Example:

for i in range(1, 6):
    print(i)

Exercise:

Print all even numbers from 1 to 20 using a loop.

Chapter 5: Functions

Functions allow code reuse. Define with def.

Example:

def greet(name):
    print("Hello", name)

greet("Fayda")

Exercise:

Create a function that returns the square of a number.
